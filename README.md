# RE:Frame XIV

A seamless, adaptive replacement HUD for Final Fantasy XIV.

RE:Frame XIV reshapes the game interface around what you are doing, keeping important information visible when it matters while reducing unnecessary screen clutter during quieter moments.

## Installation

Add the following URL as a custom plugin repository in Dalamud:

```text
https://raw.githubusercontent.com/rubyblaire/RE-Frame/main/pluginmaster.json
```

Then search for **RE:Frame XIV** in the Dalamud Plugin Installer.

## Commands

```text
/reframe
/ref
/ref edit
/ref bars
/ref keybinds
```

Opens the RE:Frame configuration window.
